# DSMovie

Semana Spring React pela DevSuperior.

Criação de uma aplicação chamada DSMovie um site que faz avaliações de filmes.

### Front-end:

- `Visual Studio Code IDE`
- `HTML`

- `TypeScript`
- `React`
- `Bootstrap`
